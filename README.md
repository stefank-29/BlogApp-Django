# Installation

Clone repository:

```
git clone https://github.com/stefank-29/BlogApp-Django.git
```

Run project:

```
cd django_project

python manage.py runserver
```

Open in browser at address:

```
http://127.0.0.1:8000/
```

-   Add new blog
